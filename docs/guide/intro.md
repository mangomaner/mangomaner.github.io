# 介绍

MangoBot 是一个基于 Java 开发的轻量级机器人框架，提供便捷的前端可视化支持，为开发者提供一个简单、灵活、易用的机器人开发平台。

## 核心特性

- **OneBot 11 协议支持**：兼容 OneBot 11 标准，支持反向 WebSocket 连接，可无缝对接 LLOneBot、NapCat 等主流 OneBot 实现。
- **插件化管理**：支持动态加载外部 JAR 插件，业务逻辑与核心框架完全解耦。
- **事件驱动**：基于注解的消息监听机制，简单直观。
- **开箱即用**：提供完善的 API 封装和工具类，极大降低开发门槛。
- **可视化**：提供浏览器界面，可直观进行各类配置和监控。


## 架构概览

MangoBot 主要由以下几个模块组成：

1.  **WebSocket 连接层**：负责与 OneBot 客户端建立连接，接收原始事件并发送 API 请求（核心）。
2.  **事件分发层**：将消息事件按优先级分发给的监听器，数据逐层传递。
3.  **插件管理层**：负责插件的加载、卸载和生命周期管理。基于主项目进行依赖注入，一行代码获取海量能力。
4.  **业务逻辑层**：开发者编写的处理器，负责具体的业务逻辑。
